# Описание проекта: justwriting.one

**justwriting.one** — это минималистичная веб-платформа для писателей, созданная для того, чтобы помочь авторам сосредоточиться на процессе письма, отслеживать свой прогресс и делиться творчеством с сообществом.

## Основная концепция
Приложение следует философии «чистого листа». Никаких лишних настроек, сложных меню или отвлекающих факторов. Только вы и ваш текст.

## Ключевые возможности

### 1. Режимы написания (Focus Modes)
*   **Секундомер**: Свободное письмо с отслеживанием времени.
*   **Таймер**: Установка цели по времени (например, писать 25 минут).
*   **Цель по словам**: Установка цели по количеству слов (например, 500 слов).
*   *Особенность*: При достижении цели сессия не прерывается, позволяя сохранить поток мыслей, но уведомляет автора об успехе.

### 2. Интеллектуальный помощник (AI-Powered)
*   Интеграция с **Google Gemini AI** для помощи в редактировании, поиске синонимов или продолжении мысли (если автор застрял).

### 3. Аналитика и прогресс
*   **Статистика в реальном времени**: Подсчет слов и скорости письма (WPM — слов в минуту).
*   **Облако слов**: Визуализация наиболее часто используемых слов в профиле.
*   **Интерактивный поиск**: Возможность найти все сессии, где использовалось конкретное слово из облака.
*   **Графики**: Визуализация активности за последние 7 дней.

### 4. Сообщество и приватность
*   **Лента (Feed)**: Возможность публиковать свои тексты в общую ленту (публично или анонимно).
*   **Личный архив**: Все сессии автоматически сохраняются в приватном архиве.
*   **Черновики**: Автосохранение текста в локальное хранилище браузера для защиты от потери данных.

## Технологический стек
*   **Frontend**: React 18, Vite (сборка), TypeScript.
*   **Стилизация**: Tailwind CSS (современный адаптивный дизайн).
*   **Анимации**: Framer Motion (плавные переходы и интерфейс).
*   **Иконки**: Lucide React.
*   **Графики**: Recharts.
*   **Backend**: Firebase (Authentication для входа, Firestore для базы данных).
*   **AI**: Google Generative AI (Gemini Pro).

## Дизайн
Приложение поддерживает **темную и светлую темы**, адаптируясь под системные настройки пользователя или ручной выбор. Интерфейс построен на принципах «воздушности» и типографики, чтобы текст был главным героем.
