Тихий редактор для свободного письма.
Один знак, одна буква, одна минута.

Бренд-гид собирает все элементы фирменного стиля justwriting в один документ: знак, лок-апы, фавикон, иконка приложения, паттерн, типографика, палитра, тон.

01

Знак

основной · на тёмном
Корпус Cartier Tank, римские цифры на 12/3/6/9, chemin de fer, заводная коронка, клавиша с «j».
основной · на светлом
На светлом фоне используется тот же знак с инверсией: тёмно-аметистовый корпус, светлая клавиша, заполненная primary-цветом.
анатомия знака
Шесть элементов
  • Корпус — 156×156, скругление R14.
  • Внутренний bezel — 136×136, R10.
  • Римские цифры — Lora 400, размер 11pt при 200×200.
  • Chemin de fer — 7 минутных штрихов на каждую сторону.
  • Клавиша — 76×76, R10, белая.
  • «j» — Inter 500, размер 56pt при 200×200.
  • Коронка — заводной винтик справа.
охранное поле
x
x

Минимальное охранное поле вокруг знака — x = 1/4 ширины корпуса. В этой зоне не должно быть посторонних элементов: текста, других логотипов, иконок.

формула
Lora пишет — Inter управляет.

Lora — голос пользователя: дневник, заметки, цитаты, заголовки, всё что про чувство и текст. Inter — голос продукта: кнопки, подписи, метрики, навигация. Никогда не смешивать в одном элементе.

07

Тон голоса

тихий

Не кричит, не подгоняет, не геймифицирует. Письмо — само по себе награда.

взрослый

Без эмодзи в продуктовых текстах, без восклицаний. Серьёзный, но не сухой.

тёплый

Обращается на «ты», использует второе лицо. Замечает усилие, но не льстит.
