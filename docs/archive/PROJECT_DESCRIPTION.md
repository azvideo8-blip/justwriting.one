# justwriting.one

**justwriting.one** — это минималистичная и функциональная платформа для осознанного письма, созданная для тех, кто ценит глубокую концентрацию и хочет превратить процесс письма в медитативный опыт.

## Основные возможности

### 1. Иммерсивный процесс письма
*   **Динамический фон**: Цвет фона плавно меняется в зависимости от вашей скорости печати (WPM), создавая визуальный ритм и обратную связь.
*   **Zen-режим**: Интерфейс автоматически скрывается, когда вы начинаете писать, оставляя вас наедине с вашими мыслями.
*   **Закрепленные мысли**: Возможность закрепить важные идеи или цитаты (через выделение текста или Alt+P), которые остаются видимыми даже в Zen-режиме.
*   **Режим печатной машинки**: Автоматическая прокрутка удерживает строку, которую вы пишете, в центре экрана.
*   **Настройка под себя**: Выбор шрифтов, размера текста и ширины рабочей области.

### 2. Управление сессиями
*   **Гибкие цели**: Установка целей по времени или количеству слов для каждой сессии.
*   **Автосохранение**: Система черновиков гарантирует, что ни одно слово не будет потеряно.
*   **Аналитика в реальном времени**: Отслеживание скорости (WPM), количества слов и времени.

### 3. Архив и организация
*   **Интерактивный календарь**: Удобная навигация по дням и месяцам для просмотра истории записей.
*   **Мощный поиск и фильтры**: Поиск по тексту, фильтрация по нескольким тегам одновременно.
*   **Система тегов**: Организация записей для легкого поиска в будущем.

### 4. Сообщество и профиль
*   **Публичная лента**: Возможность делиться своими текстами с сообществом (в том числе анонимно).
*   **Профиль автора**: Отслеживание статистики активности и достижений.

### 5. Дизайн и интерфейс
*   **Минимализм**: Ничего лишнего, только вы и текст.
*   **Темная тема**: Интерфейс по умолчанию оптимизирован для длительной работы и снижения нагрузки на глаза.
*   **Адаптивность**: Полноценная работа как на десктопе, так и на мобильных устройствах.
